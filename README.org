* Install Paru

  https://github.com/Morganamilo/paru#installation

#+begin_src sh
#! /bin/bash
sudo pacman -S --needed base-devel
git clone https://aur.archlinux.org/paru.git
cd paru
makepkg -si
cd -
#+end_src
